## 本地安装

1. 启动n8n`npx n8n`

2. 下载插件并开启编译

```shell
$ cd ~/git
$ git clone https://github.com/ruanjf/n8n-nodes-api-credentials.git
$ cd n8n-nodes-api-credentials
$ npm run dev
```

3. 安装本地依赖

```shell
$ cd ~/.n8n/nodes/
$ npm install ../../git/n8n-nodes-api-credentials
```

4. 重启`n8n`
